# Rustic Agentic — freelance WordPress & hosting website

Public, single-page marketing site for Rustic Agentic (WordPress fixes, hosting migrations, speed
optimization, builds, WooCommerce, care plans).

- **Static site, no build step** — everything is in `index.html` (HTML + embedded CSS).
- **Hosting:** GitHub Pages, branch deploy from `main` (root).
- **Contact:** `mailto:` link to `scottbaker@rusticagentic.agency` (no backend / no form server needed).

## Deploy
GitHub Pages → Settings → Pages → **Deploy from a branch** → Branch: `main` → Folder: `/ (root)` → Save.
Live at: https://entropa-chain.github.io/rusticagentic/

This repo is **solely** the public website — no other code.
